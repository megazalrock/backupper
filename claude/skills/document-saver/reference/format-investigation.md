# 調査レポート フォーマット

調査結果をまとめる際に使用するテンプレート。

## テンプレート

```markdown
# [調査対象] 調査レポート

## メタ情報

- 作成日: yyyy-mm-dd
- ドキュメント種類: 調査レポート
- 関連ファイル: [調査対象のファイルパス]

## 概要

[何を調査したか、なぜ調査したかを記述]

## 調査内容

### 調査対象

[調査したコード、API、機能などの説明]

### 調査結果

[発見した内容、構造、パターンなどを記述]

### コード例

[必要に応じてコード例を記載]

## まとめ

[調査から得られた知見や結論]

## 次のアクション

[調査結果を踏まえた次のステップがあれば記載]
```

## セクション説明

| セクション | 必須 | 説明 |
|-----------|------|------|
| メタ情報 | はい | 作成日、種類、関連ファイルを記載 |
| 概要 | はい | 調査の目的と範囲を簡潔に |
| 調査対象 | はい | 何を調査したかの詳細 |
| 調査結果 | はい | 発見した内容 |
| コード例 | いいえ | 具体的なコードがあれば |
| まとめ | はい | 調査から得られた知見 |
| 次のアクション | いいえ | 次のステップがあれば |
