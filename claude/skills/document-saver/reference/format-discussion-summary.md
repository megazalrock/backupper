# 議論まとめ フォーマット

会話で決定した内容や要件をまとめる際に使用するテンプレート。

## テンプレート

```markdown
# [議論テーマ] まとめ

## メタ情報

- 作成日: yyyy-mm-dd
- ドキュメント種類: 議論まとめ

## 概要

[何について議論・決定したかを記述]

## 背景・経緯

[なぜこの議論が必要だったか]

## 決定事項

[決定した内容を箇条書きで]

## 検討した選択肢

[検討した選択肢とその理由があれば]

## 次のステップ

[決定を受けて行うべきアクション]
```

## セクション説明

| セクション | 必須 | 説明 |
|-----------|------|------|
| メタ情報 | はい | 作成日、種類を記載 |
| 概要 | はい | 議論内容を簡潔に |
| 背景・経緯 | いいえ | 議論が必要だった理由 |
| 決定事項 | はい | 決まったことを明確に |
| 検討した選択肢 | いいえ | 他の選択肢があれば |
| 次のステップ | いいえ | 今後のアクション |
