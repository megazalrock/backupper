# 汎用 フォーマット

特定の種類に当てはまらない一般的なドキュメント用テンプレート。

## テンプレート

```markdown
# [ドキュメントタイトル]

## メタ情報

- 作成日: yyyy-mm-dd
- ドキュメント種類: その他
- 関連ファイル: [任意。関連するファイルパスがあれば記載]

## 概要

[ドキュメントの概要を1-3文で簡潔に記述]

## 内容

[メインコンテンツ。必要に応じてサブセクションに分割]

## まとめ・結論

[要点や次のアクションがあれば記載]
```

## セクション説明

| セクション | 必須 | 説明 |
|-----------|------|------|
| メタ情報 | はい | 作成日、種類、関連ファイルを記載 |
| 概要 | はい | 内容を簡潔に説明 |
| 内容 | はい | メインコンテンツ |
| まとめ・結論 | いいえ | 要点があれば |

## 適用ガイドライン

- すべてのセクションが必須ではない
- 内容に応じて適宜セクションを追加・削除する
- メタ情報と概要セクションは必ず含める
