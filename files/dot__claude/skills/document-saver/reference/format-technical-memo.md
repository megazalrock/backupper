# 技術メモ フォーマット

実装中に得た知見や気づきを記録する際に使用するテンプレート。

## テンプレート

```markdown
# [トピック] 技術メモ

## メタ情報

- 作成日: yyyy-mm-dd
- ドキュメント種類: 技術メモ
- 関連ファイル: [任意]

## 概要

[何についてのメモか簡潔に記述]

## 内容

### ポイント

[重要なポイントを箇条書きで]

### 詳細

[詳細な説明が必要な場合]

### 注意点

[気をつけるべき点があれば]

## 参考

[参考にしたドキュメントやURLがあれば]
```

## セクション説明

| セクション | 必須 | 説明 |
|-----------|------|------|
| メタ情報 | はい | 作成日、種類を記載 |
| 概要 | はい | メモの内容を簡潔に |
| ポイント | はい | 重要な点を箇条書きで |
| 詳細 | いいえ | 必要に応じて詳細説明 |
| 注意点 | いいえ | 気をつけるべき点があれば |
| 参考 | いいえ | 参考資料があれば |
